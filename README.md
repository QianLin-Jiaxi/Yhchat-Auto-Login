## 云湖账号自动登录📡
#### 此项目可以自动连接云湖wss，实现保持永久在线
#### 部署此项目后使用计划任务设置为一天触发一次即可
#### 切勿使用此功能进行非法活动！后果由使用此者承担
> **get参数key：若有内置访问密钥则请求时需附带此参数**

#### [原始版本跳转](/original-%E5%8E%9F%E5%A7%8B%E7%89%88%E6%9C%AC)
#### 若服务器环境不可用新版脚本，可尝试使用原始版本

## 开发写作规则：
#### 需保存原始可用版本

#### 返回数据：
```jsonc
{
  "success": true, // 是否成功
  "results": [
    {
      "userId": "123", // 用户ID
      "success": true, // 此账号是否成功
      "error": null // 错误信息
    }
    // 支持多个账号
  ]
}
```
